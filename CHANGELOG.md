# Changelog

## [1.0.0] — 2026-04-25

### Added
- Real-time stock price fetching via yfinance 1.3.0
- Candlestick and line chart with Plotly
- Technical indicators: SMA, EMA, RSI, MACD, Bollinger Bands
- Volume subplot on price chart
- Multi-stock comparison with normalised % return
- Company info tab: market cap, P/E, 52W high/low
- Sidebar watchlist with add/remove functionality
- Auto-refresh with configurable intervals
- Error handling with retry logic (3 attempts)
- Deployed to Streamlit Community Cloud
